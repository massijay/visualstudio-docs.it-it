# <a name="ienumdebugcodecontexts-interface"></a>Interfaccia IEnumDebugCodeContexts
Enumera i contesti di codice che corrispondono a un contesto di documento.  
  
 Oltre ai metodi ereditati da `IUnknown`, `IEnumDebugCodeContexts` interfaccia espone i metodi seguenti.  
  
## <a name="methods-in-vtable-order"></a>Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IEnumDebugCodeContexts::Next](../../winscript/reference/ienumdebugcodecontexts-next.md)|Recupera un numero di segmenti nella sequenza di enumerazione specificato.|  
|[IEnumDebugCodeContexts::Skip](../../winscript/reference/ienumdebugcodecontexts-skip.md)|Ignora un numero di segmenti in una sequenza di enumerazione specificato.|  
|[IEnumDebugCodeContexts::Reset](../../winscript/reference/ienumdebugcodecontexts-reset.md)|Reimposta una sequenza di enumerazione all'inizio.|  
|[IEnumDebugCodeContexts::Clone](../../winscript/reference/ienumdebugcodecontexts-clone.md)|Crea un enumeratore che contiene lo stesso stato dell'enumeratore corrente.|
