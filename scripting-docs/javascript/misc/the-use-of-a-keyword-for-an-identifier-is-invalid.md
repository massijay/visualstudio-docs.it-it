# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>Uso di una parola chiave per un identificatore non valido
1048: uso di una parola chiave per un identificatore non valido  
  
 È possibile utilizzare una parola chiave come nome di una variabile, funzione o altri tipi di identificatori definiti dall'utente.  
  
 Per ulteriori informazioni sulle parole chiave, vedere [parole riservate JavaScript](../../javascript/reference/javascript-reserved-words.md) e [parole riservate Future di JavaScript](../../javascript/reference/javascript-future-reserved-words.md).
