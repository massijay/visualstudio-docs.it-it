# <a name="number-expected"></a>Previsto numero
Si è tentato di richiamare il **Number.prototype.toString** o **Number.prototype.valueOf** su un oggetto di un tipo diverso da **numero**. L'oggetto di questo tipo di chiamata deve essere di tipo **numero**.  
  
### <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Richiamare solo il **Number.prototype.toString** o **Number.prototype.valueOf** metodi su oggetti di tipo **numero**.
