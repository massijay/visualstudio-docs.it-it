# <a name="mathfloor-function-javascript"></a>Funzione Math.floor (JavaScript)
Restituisce l'intero massimo minore o uguale al relativo argomento numerico.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio *numero* argomento è un'espressione numerica.  
  
 Il valore restituito è un valore intero uguale all'intero massimo minore o uguale al relativo argomento numerico.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
