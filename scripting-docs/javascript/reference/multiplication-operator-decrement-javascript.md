# <a name="multiplication-operator--javascript"></a>Operatore di moltiplicazione (*) (JavaScript)
Moltiplica il valore di due espressioni.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>Parametri  
 *risultato*  
 Qualsiasi variabile.  
  
 `number1`  
 Qualsiasi espressione.  
  
 `number2`  
 Qualsiasi espressione.
