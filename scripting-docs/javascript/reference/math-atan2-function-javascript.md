# <a name="mathatan2-function-javascript"></a>Funzione Math.atan2 (JavaScript)
Restituisce l'angolo (in radianti) dall'asse X a un punto (*y*,*x*).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parametri  
 *x*  
 Obbligatorio. Un'espressione numerica che rappresenta la coordinata x cartesiana.  
  
 *y*  
 Obbligatorio. Un'espressione numerica che rappresenta la coordinata y cartesiana.
