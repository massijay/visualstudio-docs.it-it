# <a name="global-object-javascript"></a>Oggetto Global (JavaScript)
Oggetto intrinseco il cui scopo è raccogliere le costanti e funzioni globali in un oggetto.  
  
## <a name="remarks"></a>Note  
 Il `Global` oggetto non viene mai utilizzato direttamente e non può essere creato tramite il `new` operatore. Viene creato quando viene inizializzato il motore di script, rendendo in tal modo le relative funzioni e costanti disponibili immediatamente.
