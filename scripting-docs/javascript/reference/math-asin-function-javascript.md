# <a name="mathasin-function-javascript"></a>Funzione Math.asin (JavaScript)
Restituisce l'arcoseno di un numero.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Note  
 Obbligatorio `number` argomento è un'espressione numerica per cui è necessaria l'arcoseno.  
  
 Il valore restituito è l'arcoseno del `number` argomento, in radianti.  
  
 **Si applica a**: [Math (oggetto)](../../javascript/reference/math-object-javascript.md)
