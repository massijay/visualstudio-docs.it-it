# MSBuild Error MSB3168
**MSB3168: l'elemento duplicato '\<package\>' verrà ignorato.**  
  
 Questo avviso viene generato se si è specificato che il programma di avvio automatico deve installare due package di programma di avvio automatico identici.  In questo caso, il programma di avvio automatico installerà una sola istanza.  
  
### Per correggere l'errore  
  
-   Per ulteriori informazioni sulla causa e le possibili soluzioni di un errore di MSBuild, vedere [Additional Resources](../msbuild/additional-msbuild-resources.md).
