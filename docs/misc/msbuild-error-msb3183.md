# MSBuild Error MSB3183
**MSB3183: il riferimento '\<assembly\>' è un assembly di interoperabilità che richiede attendibilità totale.**  
  
 Questo avviso viene generato quando tra i riferimenti viene incontrato un assembly di interoperabilità nella generazione del manifesto per un'applicazione parzialmente attendibile.
