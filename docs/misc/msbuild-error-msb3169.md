# MSBuild Error MSB3169
**MSB3169: errore durante la generazione di un programma di avvio automatico: '\<errore\>'**  
  
 Consente di visualizzare gli errori che non vengono individuati dalla compilazione di un programma di avvio automatico.  
  
### Per correggere l'errore  
  
-   Per ulteriori informazioni sulla causa e le possibili soluzioni di un errore di MSBuild, vedere [Additional Resources](../msbuild/additional-msbuild-resources.md).
