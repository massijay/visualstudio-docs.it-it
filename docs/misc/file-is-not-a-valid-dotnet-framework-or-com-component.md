# File is not a valid .NET Framework or COM component
Si sta tentando di visualizzare nel Visualizzatore oggetti un file che non contiene un oggetto valido.  Nel Visualizzatore oggetti è possibile visualizzare solo componenti COM o [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)].  
  
### Per correggere l'errore  
  
1.  Se si è selezionato un file non corretto, selezionare il file corretto e riprovare.  
  
2.  Se si è selezionato per errore il Visualizzatore oggetti nella finestra di dialogo **Apri con**, scegliere un programma diverso per aprire il file.
