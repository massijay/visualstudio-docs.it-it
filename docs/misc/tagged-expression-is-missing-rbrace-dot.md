# Tagged expression is missing &#39;}&#39;.
Generalmente questo errore si verifica quando in una stringa di ricerca si specifica una parentesi di apertura, \(`{`\) ma si omette la parentesi di chiusura \(`}`\).  
  
### Per correggere l'errore  
  
1.  Per cercare il valore letterale di carattere `{`, immettere `\{`.  
  
2.  Per rendere un'espressione con tag, immettere la parentesi `}` mancante.
