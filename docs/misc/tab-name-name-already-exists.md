# Tab name &lt;name&gt; already exists.
Ad ogni scheda della Casella degli strumenti deve essere assegnato un nome univoco.  Generalmente questo errore si verifica quando il nome immesso per la scheda della Casella degli strumenti corrisponde a quello di una scheda esistente nella Casella degli strumenti.  
  
### Per correggere l'errore  
  
1.  Immettere un nome univoco per la scheda.
