# Project type is not valid.
Generalmente questo errore si verifica quando si immette un valore errato per l'argomento *nomemodello* del comando **Aggiungi nuovo progetto**.  
  
### Per correggere l'errore  
  
1.  Controllare nella documentazione la sintassi corretta del comando **Aggiungi nuovo progetto**, quindi riprovare.
