# CA0053
Impossibile caricare 1 assembly aggiuntivo.  
  
 Indica che non è stato possibile caricare un assembly della regola mediante reflection.  Ciò può accadere se l'utente dispone di regole personalizzate non compilate correttamente o se lo strumento di analisi non è in grado di caricare l'assembly per altre ragioni \(ad esempio, se l'accesso al file system è stato negato\).
