# The &lt;name&gt; switch requires a value.
Generalmente questo errore si verifica quando si specifica un'opzione senza un valore obbligatorio.  
  
### Per correggere l'errore  
  
1.  Immettere di nuovo il comando, fornendo un valore per l'opzione.  
  
     \-oppure\-  
  
     Immettere di nuovo il comando, omettendo l'opzione.
