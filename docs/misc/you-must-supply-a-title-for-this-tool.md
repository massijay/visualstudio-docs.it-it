# You must supply a title for this tool.
Generalmente questo errore si verifica quando si lascia vuoto il campo **Titolo** nella finestra di dialogo **Strumenti esterni**.  
  
### Per correggere l'errore  
  
1.  Immettere un nome per il nuovo strumento esterno nella casella di testo **Titolo**.
