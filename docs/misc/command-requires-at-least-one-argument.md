# Command requires at least one argument.
Generalmente questo errore si verifica quando non si immette alcun argomento per un comando che ne richiede almeno uno.  
  
### Per correggere l'errore  
  
1.  Controllare la sintassi corretta del comando nella documentazione, quindi riprovare.
