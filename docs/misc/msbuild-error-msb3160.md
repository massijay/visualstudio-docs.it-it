# MSBuild Error MSB3160
**MSB3160: avviso di convalida XML nel file '\<file\>': '\<errore\>'**  
  
 Questo avviso viene generato quando il file del prodotto o package.xml non è conforme allo schema XML.  La seconda parte dell'avviso è uno specifico messaggio di avviso.  
  
### Per correggere l'errore  
  
-   Aggiornare il file XML per correggere l'errore di convalida.
