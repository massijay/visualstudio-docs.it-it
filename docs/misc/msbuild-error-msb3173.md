# MSBuild Error MSB3173
**MSB3173: impossibile scrivere il manifesto '\<file\>'. '  \<errore\>'**  
  
 Questo errore viene generato quando il processo di compilazione rileva un problema generale durante la scrittura di un file manifesto.  
  
### Per correggere l'errore  
  
-   Per ulteriori informazioni sulla causa e le possibili soluzioni di un errore di MSBuild, vedere [Additional Resources](../msbuild/additional-msbuild-resources.md).
