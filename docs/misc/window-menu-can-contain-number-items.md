# Window menu can contain &lt;number&gt; items.
Nell'elenco di finestre del menu Finestra possono essere visualizzati al massimo 24 nomi di finestra.  Generalmente questo errore si verifica quando si immette un numero minore di 1 o maggiore di 24 nella casella **Il menu Finestra contiene \<numero\> di elementi** nel riquadro Generale dell'opzione Ambiente nella finestra di dialogo Opzioni.  
  
### Per correggere l'errore  
  
1.  Immettere un numero compreso tra 1 e 24 nella casella.
