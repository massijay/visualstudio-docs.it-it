# Command does not accept any arguments.
Generalmente questo errore si verifica quando si specifica un argomento per un comando che accetta solo opzioni.  
  
### Per correggere l'errore  
  
1.  Controllare la sintassi corretta del comando nella documentazione, quindi riprovare.
