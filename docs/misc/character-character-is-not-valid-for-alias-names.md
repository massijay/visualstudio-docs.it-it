# Character &lt;character&gt; is not valid for alias names.
Generalmente questo errore si verifica quando si immette un carattere non valido come parte di un nome alias.  I nomi alias non possono contenere uno spazio vuoto, \=, \[, \], \/, \- o ".  
  
### Per correggere l'errore  
  
1.  Immettere di nuovo il nome alias, omettendo il carattere non valido.
